Examples
========

These examples can be found in the ``<nengo_dl>/docs/examples`` directory
(where ``<nengo_dl>`` is the location of the NengoDL package).  The examples
are IPython/Jupyter notebooks; if you would like to run them yourself, refer to
the
`Jupyter documentation <http://jupyter-notebook.readthedocs.io/en/latest/>`_.
